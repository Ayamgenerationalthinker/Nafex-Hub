## 🎯 OBJECTIVE

---

## 📊 FEATURE: ANALYTICS DASHBOARD

### New Route:

* /dashboard/analytics

### Access:

* Only logged-in business owners

---

## 📈 METRICS TO TRACK

For each business:

1. Profile Views
2. Messages Received
3. Orders Created
4. Conversion Rate (orders ÷ views)
5. Growth over time (daily & weekly)

---

## 🗄️ DATABASE STRUCTURE

Create a new collection:

### AnalyticsEvent

* id
* businessId
* userId (optional)
* type (view | message | order)
* createdAt

---

## ⚙️ EVENT TRACKING LOGIC

Automatically log events:

* When a user visits a business profile → create event (type: "view")
* When a message is sent → create event (type: "message")
* When an order is placed → create event (type: "order")

---

## 📊 BACKEND AGGREGATION

Create API:

GET /api/analytics/business/:businessId

* totalViews
* totalMessages
* totalOrders

* dailyStats (last 7 days)
* weeklyStats
